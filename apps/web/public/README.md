# Favicon and Logo Assets

This directory contains the favicon and logo assets for CodeSenseiSearch.

## Required Assets:

### Favicons
- `favicon.ico` - 16x16, 32x32, 48x48 favicon
- `icon-16.png` - 16x16 PNG icon
- `icon-32.png` - 32x32 PNG icon
- `apple-touch-icon.png` - 180x180 Apple touch icon

### Open Graph Images
- `og-image.png` - 1200x630 Open Graph image
- `og-image-home.png` - 1200x630 Home page specific OG image
- `og-image-search.png` - 1200x630 Search page specific OG image

### Logos
- `logo.png` - 512x512 main logo
- `logo-dark.png` - 512x512 dark theme logo
- `logo-light.png` - 512x512 light theme logo

### Web App Manifest Icons
- `icon-192.png` - 192x192 PWA icon
- `icon-512.png` - 512x512 PWA icon

## Notes:
- All PNG files should be optimized for web
- SVG versions should be provided where possible
- Dark/light theme variants ensure proper contrast
- Open Graph images should include branding and be readable at small sizes
